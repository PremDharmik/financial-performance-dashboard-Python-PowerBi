# Financial Performance Dashboard - PowerBi - Python
> Interactive Power BI dashboard for analyzing financial metrics and EV market trends

## 📖 Overview
This project focuses on building a comprehensive Power BI dashboard to analyze global financial performance and extract actionable business insights. It transforms raw financial data into a structured, visual format, enabling stakeholders to understand the relationships between key metrics such as Sales, Profit, Cost of Goods Sold (COGS), and Discounts.

The dashboard is designed as a centralized decision-making tool, allowing users to explore trends, identify performance drivers, and uncover areas of improvement. Through interactive visualizations, it supports strategic planning by highlighting growth patterns, regional performance, and profitability dynamics.

In addition to financial analysis, the project also includes an overview of India’s electric vehicle (EV) market, offering insights into market leadership, category performance, and growth trends from 2014 to 2024.

---

## ✨ Features
- KPI cards for **Sales, Profit, and Margin**
- Sales trend analysis with forecast comparison
- Year-over-Year (YoY) growth analysis
- Geographic performance visualization
- Product profitability treemap
- Segment-wise sales breakdown
- Detailed performance matrix for granular insights
- Discount vs. profitability scatter plot
- Discount distribution across customer segments
- EV market analysis with state-wise and category insights
- Identification of high-performing and underperforming products

## 📊 How It Works
The dashboard is structured into multiple analytical views:

- **Executive Summary:**  
  Displays key KPIs, sales trends, and geographic insights to provide an instant snapshot of business performance.

- **Product & Segment Analysis:**  
  Uses treemaps and matrices to evaluate product profitability and segment contribution, helping identify top-performing and underperforming areas.

- **Discount Analysis:**  
  Examines the relationship between discount strategies and profitability using scatter plots and composition charts, enabling data-driven pricing decisions.

- **EV Market Analysis:**  
  Tracks India’s EV adoption trends, highlighting leading states, dominant vehicle categories (2-wheelers and 3-wheelers), and growth patterns over time.

Together, these components create an interactive system for exploring data, identifying insights, and supporting strategic decisions.
